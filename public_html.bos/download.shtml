Development version

The game is still in development. You can get the latest version with subversion: 
svn co svn://bos.seul.org/svn/bos/data.bos/trunk data.bos

You'll need the latest development version of the stratagus engine:

svn co https://svn.sourceforge.net/svnroot/stratagus/stratagus/trunk stratagus

To start the game run :

./stratagus -d path/to/data.bos

See bos FAQ to get more informations.

You can browse the source code via websvn.

Old development done before december 17th 2004 can be found in the CVS repository.
