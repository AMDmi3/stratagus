Invasion - Battle of Survival (BoS) 2.0 soon to be released

With the Stratagus engine 2.2 release coming in sight, Invasion Team is pleased to announce the BoS 2.0 freezing process.

BoS 2.0 integrates new features like Guichan menus, improved UI, image-based map system, new units, new set elements... (and a lot of new, great, and funny bugs from Stratagus team!! :o)
