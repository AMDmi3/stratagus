About

Invasion - Battle of Survival is a real-time strategy game using the Stratagus game engine.

News

Invasion - Battle of Survival 2.0.1 has been released !

Get it from the download section.

Invasion - Battle of Survival 2.0 integrates new features like Guichan menus, improved UI, image-based map system, new units, new set elements, internationalisation, ... Version 2.0.1 fixes a few bugs.
