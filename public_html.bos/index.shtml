About

Bos Wars is a real-time strategy game that uses a modified version of the Stratagus game engine.

Bos Wars is copyrighted in 2004-2007 by Tina Petersen Jensen, François Beerten et al. See COPYRIGHT.txt in the package for the full copyright notice.

News

Feb 9, 2007 - Project renamed to Bos Wars.

Invasion - Battle of Survival has been renamed to Bos Wars due to popular demand.

Merge of the Stratagus engine into the BOS project.

To speed up development, the Stratagus engine has been merged into the BOS project. Due to the merge, the subversion URL has changed. You can find all details in the development page.

Invasion - Battle of Survival 2.0.1 has been released !

Get it from the download section.

Invasion - Battle of Survival 2.0 integrates new features like Guichan menus, improved UI, image-based map system, new units, new set elements, internationalisation, ... Version 2.0.1 fixes a few bugs.
