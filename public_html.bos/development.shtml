Before you start

Before any development, feature request or bug report, you should get and use the development version of the Stratagus engine and of Bos. The Download section explains how to get the development version of Bos.

Before starting a new development, contact us to check if it fits in the game and if nobody else is working on it.

New Features

The design of the game happens on irc and on the Bos Wiki. Don't hesitate to add your comments and ideas.
