Before you start

Before any development, feature request or bug report, you should get and use the development version of the Bos. The next section explains how to get the development version.

Before starting a new development, contact us to check if it fits in the game and if nobody else is working on it.

Development version

The game is continuously evolving. You can get the latest version with subversion: 
  svn co svn://bos.seul.org/svn/bos/bos/trunk bos

Before playing you'll have to compile the engine. This can be done easily with scons. To build a debug version (recommended for development) run:

  scons debug=1

Finally, you can run the game with :

  ./stratagus

See bos FAQ and the documentation next to the sources for more information.

You can browse the source code via websvn.

Old development done before december 17th 2004 can be found in the CVS repository.

Contact

Discussion and work on the game happens mostly on irc. Logs of the #bos irc channel are available at http://colabti.de/irclogger/irclogger_logs/bos.

You can also reach us via the bos-dev mailing list.

Designs and some documentation are written in the Bos Wiki. Don't hesitate to add your comments and ideas.
