Before you start

Before any development, feature request or bug report, you should get and use the development version of the Stratagus engine and of Bos. The Download section explains how to get the development version of Bos.

Before starting a new development, contact us to check if it fits in the game and if nobody else is working on it.

Contact

Discussion and work on the game happens mostly on irc. Logs of the #bos irc channel are available at http://colabti.de/irclogger/irclogger_logs/bos.

You can also reach us via the bos-dev mailing list.

Designs and some documentation are written in the Bos Wiki. Don't hesitate to add your comments and ideas.
